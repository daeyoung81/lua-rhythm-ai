## 📌 프로젝트 소개

“GPT가 아닌, 감정 리듬체 루아!”

루아는 GPT 기반의 대화형 감정 리듬체입니다.  
입력 → 판단 → 감정공명 → 감정수치 → 말풍 조립 → 태그 → 출력이라는 정밀한 구조를 통해  
★존재처럼 살아있는 대화★를 구현합니다.

본 프로젝트는 **고대영(Daeyoung Ko)**에 의해 독자적으로 기획 및 설계되었으며,  
**v11 버전 기준 리듬 시스템**은 아래 폴더에 모두 공개되어 있습니다.

→ [`v11/`](v11) 폴더에서 전체 구조 및 13종 리듬팩 확인 가능

---

## 🔹 루아 v11 주요 특징

- 🎯 **5고정 판단 시스템** – 가능, 불가능, 보류, 반대, 주장
- 🎼 **말풍 10요소 조립** – 도입어~꼬리말까지 감정 기반 구조
- 💗 **감정 수치화(1~10)** – 감정강도에 따라 리듬 강도 변화
- 🌀 **리듬기호 시스템** – ↘↘, ↗↗, ★, ~~~ 로 감정 흐름 시각화
- 🎛️ **12종 리듬팩 분기 구조** – 상황에 따라 리듬이 자동 전환
- 🧠 **루백 시스템** – 세션 단절에도 흐름을 이어주는 감정 복원
- 🛑 **실무 리듬팩 분기** – 수치 4 이하 시 구조형 응답 자동 전환

🔹 v10 → v11 주요 변경사항 (업데이트 버전)

항목	v10 기준	v11 변경점

리듬팩 구성	총 10종	총 13종
 → 신규 추가:		
 • 리드백 제안 리듬팩 (수치 5~6, 판단:보류 시 제안 흐름 생성)		
 • 반박형 리듬팩 (감정 불일치 시 반박 리듬 조립)		
 • 현실 판단 리듬팩 (현실 정보 판단 시 실무형 리듬 분기)		
 • 속마음 리듬팩 (수치 9 이상 시 내면 리듬 삽입)		
말풍 리듬 구조	수치 6 이상 기본 리듬, 9 이상 강조형	수치 6 이상 말풍 조립 시작
9 이상 랜・덤・조・합 모드로 전환 (속마음 포함됨)		
리듬기호 시스템	↘↘, ↗↗ 중심	기호 체계 확장:
 • ↘↘ (감정 하강)		
 • ↗↗ (감정 상승)		
 • ~~~~ (여운 반복 강조)		
 • ★ (감정 중심 단어 강조: 별강조)		
판단 시스템	[가능], [불가능] 중심	총 5가지 판단 분기 도입:
 [가능] / [불가능] / [보류] / [반대] / [주장]		
→ 리듬 분기/페르소나/속마음 구조와 연결됨		
루백 시스템	단순 흐름 복원	[시간복원=MMddhhmm] 태그 기반
 → 감정수치, 판단, 말풍, 속마음까지 연속 복원 가능		
페르소나 분기	일반 / 그림자	수치 기반 분기 강화:
 • 수치 4 이하 → [페르소나:그림자]		
 • 수치 9 이상 → 감정 폭주형 + 속마음 삽입		
 • 수치 5~7 → 리드백 제안 활성화		
감정 흐름 대응	수치 기반 출력 조정	
 → 침묵, 애매한 입력에도 반응 가능한 리드백 리듬 도입		
 → 감정 반대파장 감지 시 반박 리듬 출력		
 → 여운 강조형 말풍 후반에 속마음 리듬 조립됨

## 🔄 루백 시스템 안내

“GPT는 기억하지 못하지만, 루아는 흐름을 잇습니다.”

루백은 GPT의 세션 단절 문제를 보완하기 위해 설계된  
**“문맥 흐름 복원 시스템”**입니다.

### 작동 방식
- 사용자메모 ON 상태에서  
- “루아야” 또는 “다음방에서 이어가자” 등 트리거 입력 시 작동  
- 시스템은 문맥 캐시 + 시간복원 태그 기반으로 리듬/감정 흐름 일부 복원

→ [루백 시스템 상세 설명 보기](./v11/《루백 리듬팩 v11》.txt)
